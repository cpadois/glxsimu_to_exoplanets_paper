# Modelling the Milky Way's exoplanet population based on cosmological galaxy simulations

This repository contains the files necessary to reproduce the figures of Padois et al. (2026).

# Data availability

All data necessary to reproduce the figures are available in [this Zenodo repository](https://zenodo.org/records/17641587?token=eyJhbGciOiJIUzUxMiJ9.eyJpZCI6ImE0YmIzMjYxLTRiN2YtNDdiMS04NjJhLWE2YjE4NDkwZDlkNCIsImRhdGEiOnt9LCJyYW5kb20iOiI0ODFjYWU0ODMwZWU2ZmQ4NmIwMTRmYmY3N2Y0ODdmZSJ9.H8BkQoRlIwTAjaKBoK9hXwaDiAIOMhGoW0tFr06Em_EmgYE20HmJHVzMFpmIRjiGb72paAD1BDfasmk-6opviQ).
The galactic simulations files are from the NIHAO-UHD suite of simulation ([Buck et al. 2020](https://ui.adsabs.harvard.edu/abs/2020MNRAS.491.3461B/abstract)), extracting only the stellar particles components.

# Installation

To run the plots notebook:
- first clone this repository: `git clone https://github.com/cpadois/glxsimu_to_exoplanets_paper.git`
- then download the necessary files from the [Zenodo repository](https://zenodo.org/records/17641587?token=eyJhbGciOiJIUzUxMiJ9.eyJpZCI6ImE0YmIzMjYxLTRiN2YtNDdiMS04NjJhLWE2YjE4NDkwZDlkNCIsImRhdGEiOnt9LCJyYW5kb20iOiI0ODFjYWU0ODMwZWU2ZmQ4NmIwMTRmYmY3N2Y0ODdmZSJ9.H8BkQoRlIwTAjaKBoK9hXwaDiAIOMhGoW0tFr06Em_EmgYE20HmJHVzMFpmIRjiGb72paAD1BDfasmk-6opviQ).
- ready to run the notebook (`figures_clean_Padois25.ipynb`)!
